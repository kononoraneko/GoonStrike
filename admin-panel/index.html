GoonStrike Registry Admin

Dev admin panel for server credential provisioning and online server monitoring.

Connection

Quick provision (secret shown once)

Generates server_id/key_id/secret on the backend. Copy the secret immediately; it is not stored in plaintext.


    


    

      
Enrollment token (dedicated auto-fetch)

      
Mint a one-time token. Start the dedicated server with --registry-enroll-token and --backend-url; it exchanges the token for keys and saves them (default user://goonstrike_dedicated_registry.cfg).

      

        
        
      

      

        
      

      

    


    

      
VDS orchestrator

      

        Requires backend env GOONSTRIKE_ORCHESTRATOR_URL and matching secrets. Calls
        POST /servers/admin/orchestrator/spawn → node agent runs docker run.
        Backend URL for container must be reachable from inside the dedicated container (often your public IP or domain, not http://backend:8000).
